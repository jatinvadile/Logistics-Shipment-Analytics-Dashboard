# Logistics-Shipment-Analytics-Dashboard

Analyzed shipment lifecycle and delivery performance across 7 global hubs (Mumbai, Dubai, Italy, China, Indianapolis, Memphis) to improve On-Time In-Full (OTIF) delivery and optimize logistics efficiency.

1. Built an end-to-end data pipeline using SQL, Python, Snowflake, and Power BI - integrating live shipment tracking data via FedEx REST APIs across all global hubs.
2. Developed advanced logic to track shipment routes, transit times, delay patterns, and first-attempt delivery success - including weekday vs. Saturday delivery scenarios.
3. Identified key bottleneck hubs and delay-prone routes, improving route visibility and enabling proactive decision-making - achieving 20% improvement in delivery planning efficiency.
4. Designed interactive Power BI dashboards featuring delay heatmaps, hub-wise performance scorecards, and shipment flow tracking.
5. Reduced manual shipment tracking effort by 50%, delivering real-time operational insights to logistics managers across regions.
